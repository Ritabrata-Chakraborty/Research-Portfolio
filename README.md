# Ritabrata Chakraborty — Research Portfolio

Personal research portfolio site built with React, TypeScript, and Vite.

**Live site:** https://ritabrata-chakraborty.github.io/Research-Portfolio/

---

## Sections

- **About** — Bio, contact links, and location
- **Education** — UPenn (M.S.E. Robotics) and BITS Pilani (B.E. Mechanical Engineering)
- **Experience** — Research and industry positions
- **Publications** — Peer-reviewed papers and preprints
- **Projects** — Filterable by tag (Robotics, Vision, Simulation, Gen-AI, CAD/FEM)
- **Achievements** — Awards, scholarships, and leadership
- **Blogs** — Writing and articles
- **Skills** — Technical skills overview

---

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion (`motion/react`)
- Lucide React icons

---

## Run Locally

**Prerequisites:** Node.js

```bash
npm install
npm run dev
```

Open http://localhost:5173

---

## Deploy

The site is deployed via GitHub Pages. To build:

```bash
npm run build
```

Output goes to `dist/`. Push to `master` and GitHub Pages serves it automatically.
